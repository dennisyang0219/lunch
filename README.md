# 🍱 團購訂便當系統 (Lunchbox Ordering System)

這是一個專為團隊、辦公室設計的**點餐與統計系統**。包含前台使用者點餐介面 (`index.html`) 與後台管理者設定介面 (`admin.html`)，並結合 **Google Sheets (Google 試算表)** 作為資料儲存庫。

---

## 🚀 快速安裝與設定步驟

系統完全基於靜態網頁（不需安裝 node/npm 或是後端伺服器），只需要與 Google Sheets 進行串接即可運作。請按照以下步驟完成設定：

### 第一步：建立 Google 試算表與設定 Apps Script

1. 開啟 [Google 雲端硬碟](https://drive.google.com/)，建立一個全新的 **Google 試算表**。
2. 點選上方選單的 **「擴充功能」** -> **「Apps Script」**。
3. 清空編輯器中原本的所有內容。
4. 打開本專案資料夾下的 [google-apps-script.js](file:///c:/order/google-apps-script.js) 檔案，複製全部程式碼，並貼入 Apps Script 編輯器中。
5. 點擊編輯器上方的 💾 (儲存) 按鈕。

### 第二步：將 Apps Script 部署為網頁應用程式 (Web App)

1. 點擊 Apps Script 視窗右上角的 **「部署」** -> **「新增部署作業」**。
2. 點擊左上角「選取類型」旁的 ⚙️ 齒輪，選擇 **「網頁應用程式」**。
3. 進行以下設定：
   * **說明**：隨意填寫（例如：*訂便當系統 API*）。
   * **專案執行身分**：選擇 **「您的帳戶」** (通常顯示為您的 Gmail 帳號)。
   * **誰有存取權**：選擇 **「所有人 (Anyone)」** (※這是關鍵！否則前台網頁無法成功傳送資料)。
4. 點擊下方 **「部署」**。
5. （首次部署）系統會跳出授權視窗，請點選 **「授予存取權」**，選擇您的 Google 帳戶。如果顯示「Google 尚未驗證此應用程式」，請點擊左下角 **「進階」** -> **「前往『未命名專案』(不安全)」**，並同意授予權限。
6. 部署完成後，複製畫面上顯示的 **「網頁應用程式 URL」** (格式為 `https://script.google.com/macros/s/.../exec`)。

### 第三步：連結網頁與設定

1. 在瀏覽器中開啟 [index.html](file:///c:/order/index.html)。
2. 點擊右上角 **「管理者後台」** 按鈕。
3. 系統將提示輸入密碼，請輸入：`water603`。
4. 跳轉至 `admin.html` 後，在 **Google Sheets API 設定** 欄位中，貼上剛才複製的「網頁應用程式 URL」。
5. 點擊 **「儲存」** 按鈕。系統將自動與您的 Google Sheet 進行連線，並自動在試算表中建立 `Config`、`Menu`、`Orders` 三個工作表。

## 🛠️ 管理者使用指南 (`admin.html`)

後台提供以下三大核心管理功能：

1. **設定截止時間**：
   * 選取日期與時間後，點擊 **「儲存時間」**。前台 `index.html` 將會自動顯示倒數計時。
   * 若剩下不到 15 分鐘，前台時間會自動變紅並閃爍警示。
   * 時間一到，前台將自動關閉點餐功能並顯示「已截止」。
   * 若不需要截止限制，可點擊 **「清除截止時間」**。

2. **管理店家與便當品項**：
   * 點擊 **「新增」** 來增加便當店家。
   * 選擇左側店家後，即可在右側編輯該店的便當名稱與價格。
   * 編輯完畢後，**務必點擊最下方的「同步菜單到 Google Sheet」**，系統會將最新的菜單同步寫入 Google Sheet。

3. **清除目前所有訂單 (危險區域)**：
   * 點擊 **「清空 Google Sheet 所有訂單」** 可以一鍵清除試算表中的所有訂餐紀錄，以便開啟下一波新點餐。

## 🍱 使用者點餐指南 (`index.html`)

使用者可以在主畫面進行點餐：

1. **輸入姓名**：輸入框會自動記憶上一次輸入的名字，免去每次重複輸入的麻煩。
2. **選擇店家**：下拉選單選取今天想吃的店家，系統會自動載入該店家的菜單品項。
3. **選擇品項**：支援**複選**（如果一個人要幫同事點，或者想吃多個品項，都可以直接勾選）。
4. **即時訂單統計**：
   * 畫面下方會即時顯示**所有人目前已點的品項與數量**（例如：*排骨飯 5份、雞腿飯 3份*），並附上點餐人的姓名。
   * 點餐完畢送出後，統計資料會自動重新整理；也可以手動點擊「重新整理」以取得最新狀態。
